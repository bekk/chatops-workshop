#production_deploy

#logs

#feedback

Jeg vil vite når ... Vi får en feilmelding fra sluttbrukere

#sukker-deploy

Jeg vil gjøre ... en deploy

#sukker-deploy

Jeg vil gjøre ... en deploy og jeg er helt ny på prosjektet (det kan du for du har jo sett masse folk gjøre en deploy tidligere i kanalen)

#random

Jeg vil være med på morroa ... bli med i konkurransen vår (en implementasjon av spis-mest-mulig-miljøvennlig-konkurransen på prosjektet) Dette er fin måte å bygge litt kultur og ikke mist for å oppfordre flere til å ta i bruk, kode på og se muligheter med boten.

Hubot

  • Chatbot laget av github
  • Skrevet i Coffescript og Node.js
  • Første og største av sin type
  • Enkel å konfigure
  • Enkel å deploye. Ferdig oppsett til heroku
  • Enkel å integrere med chats (Adaptere)
  • Enkel å lage scripts til
  • Eller finne ferdige scripts på nett

    • Arkitektur

    ```js // Description: // Et enkelt testscript for enkel respond-funksjonalitet // // Commands: // hubot hello - boten hilser tilbake module.exports = function (robot) { robot.respond(/hello/, function(res){ res.reply("Velkommen til workshop!"); }) } ```
